QUICK START (4 STEPS)
================================================================================

1. START SERVER
   $ npm run dev
   → Server runs at http://localhost:3000

2. SELECT BAND
   → Click one of 5 frequency band buttons
   → Carrier frequency auto-updates
   → Default: Theta (meditation)

3. PICK DURATION
   → Choose: 15, 30, 45, or 60 minutes
   → Default: 30 minutes

4. START SESSION
   → Click "Start [Duration] Minute Session"
   → Watch timer count down
   → Pause/Resume as needed
   → Session saves automatically when complete
